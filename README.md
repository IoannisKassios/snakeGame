# 🐍 Snake Game - Python Edition
Classic Snake Game built in Python.

## 🎮 Description
A fun and simple implementation of the classic Snake game, written in Python. The game features:

* Smooth keyboard-controlled movement
* Score tracking as the snake grows
* Self-collision and wall boundary detection
* Increasing difficulty as the snake gets longer
* Clean and beginner-friendly code structure

Great for learning Python game logic, loops, and basic event handling.

---

## 🚀 Getting Started

### Prerequisites

* Python 3.7 or later
* `pygame` library (if applicable)

### Install Dependencies

```bash
pip install pygame
```

### Run the Game

1. Clone the repository:

```bash
git clone https://github.com/IoannisKassios/snakeGame.git
cd snakeGame
```

2. Run the game:

```bash
python snake.py
```

3. Use the **arrow keys** to control the snake. Avoid walls and yourself!
